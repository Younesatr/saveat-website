## ğŸš€ How to Run

### Method 1: Simple (Browser)
Since this project uses modern ES modules and Tailwind via CDN for rapid prototyping:
1.  Simply open `index.html` in a modern web browser.
2.  **Note**: For the AI Chat to work, you may need a local server due to browser security policies (CORS).

### Method 2: Developer (Node.js)
If you want to develop further using the included build system:

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the local server:
    ```bash
    npm run dev
    ```
3.  Build for production:
    ```bash
    npm run build
    ```
